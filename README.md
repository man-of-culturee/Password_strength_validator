# Password Check(GUI)
This is a Python GUI Application that checks password strength based on multiple security rules.

## Features
- Real-time password strength checking
- Interactive GUI (Using Tkinter)
- Strength scoring system
- Visual progress bar with color indicators
- Detailed feedback for weak passwords
- Password visibility toggle

-Make sure u have python 3.8+ installed
-clone this repository: https://github.com/man-of-culturee/password_strength_validator.git
